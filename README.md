# myportfolio

A beautiful, modern, interactive desktop-style personal portfolio application built with React, Tailwind CSS, and Framer Motion. 
It features a unique windowed UI, where each application module (About Me, Projects, Contact) is movable, resizable, and distinct.

## Features
- **Windowed UI Simulator:** Drag, minimize, maximize, and close various "apps".
- **Dynamic Theming:** Smoothly transitions gradient colors depending on the active top window.
- **Glassmorphism Design:** Employs vibrant pastel themes and subtle glass UI effects.

## Technologies
- **React.js** (Frontend UI)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling & Animation)
- **Framer Motion** (Interactions & Window physics)
- **Zustand** (Global Application State Management)
- **Lucide React** (Icons)

## Getting Started

To run this application locally:

1. **Install dependencies:**
   \`\`\`bash
   npm install
   \`\`\`

2. **Start the development server:**
   \`\`\`bash
   npm run dev
   \`\`\`
   Your app will be available at http://localhost:3000

3. **Build for production:**
   \`\`\`bash
   npm run build
   \`\`\`
   4. live demo
      myportfolio-two-eosin.vercel.app
